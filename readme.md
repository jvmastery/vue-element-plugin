# 使用
## 1、引入依赖
npm install @jvmaster/f-ui

## 2、使用
### 2.1 全局引入
```
import Fui from '@jvmaster/f-ui'
const app = createApp(App)

app.use(Fui)
```

### 按需引入
```
import { FForm } from '@jvmaster/f-ui'

```
